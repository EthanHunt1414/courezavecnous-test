## 🆘 Troubleshooting

### Images not showing?
**Problem:** Broken image icons appear
**Solution:** 
- Check that `/images/` folder was uploaded
- Verify file names match exactly (case-sensitive)
- Ensure folder structure: `/images/freyming.jpg` not `/freyming.jpg`

### Layout looks broken?
**Problem:** No styling, just plain text
**Solution:**
- Ensure `styles.css` is in same folder as `index.html`
- Check file uploaded correctly (not corrupted)
- Clear browser cache (Ctrl+F5)

### Links not working?
**Problem:** Clicking doesn't do anything
**Solution:**
- Ensure `script.js` is uploaded
- Check browser console for errors (F12)
- Try different browser
